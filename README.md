# Youtube-Audio-CutBySubtitle
##從播放清單下載youtube的影片與切割語音片段 
需安裝套件pytube、youtube-dl
# 需安裝套件

引有的套件有:

	pip install pytube
	pip install youtube-dl
	pip install webvtt-py
	pip install pydub
	pip install progressbar
# 遇到的bug

ffprobe or avprobe not found. Please install one
